> 思考：百度地图mapv开发流程是怎样的？

::: details
1. 引入 js 库
    - 百度地图 js
    - 百度地图 mapv
    - 百度地图 mapvgl
2. 编写容器组件
3. 初始化 Map 对象
4. 初始化绘制数据，数据格式为：
```js
[{
  geometry: {
    type: 'Point',
    coordinates: [x, y]
  },
  properties: {
    count: Math.random() * 100
    // other properties...
  }
}, {
  // other data...
}]
```
5. 初始化 mapvgl.View 图层管理器
6. 初始化 mapvgl.Intensity 数据显示强度
7. 初始化 mapvgl.PointLayer 图层
8. 调用 view.addLayer 添加图层到图层管理器
9. 调用 mapvgl.PointLayer.setData 关联图层和数据
:::
